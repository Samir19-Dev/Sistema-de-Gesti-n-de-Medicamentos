* Samir Julio

## 📌 Descripción

Aplicación desarrollada en Java con interfaz gráfica (Swing) que permite gestionar un inventario de medicamentos y sus tipos.

---

## 🧠 Funcionalidades

### Medicamentos

* Agregar
* Buscar por código
* Actualizar
* Eliminar
* Listar en tabla

### Tipos de Medicamentos

* Agregar
* Buscar
* Actualizar
* Eliminar

### Sistema

* Guardado en archivos `.txt`
* Carga automática de datos
* Validaciones
* Estadísticas (stock, bajo stock, total)
